# TareaCorta3
Primero, se inicia el proyecto donde se encuentra la API utilizando el siguiente comando en la terminal:

cd ./NodeApi
node ./server.js

Después, se abre otra terminal, nos ubicamos en el proyecto de Angular y se inicia la aplicación:

cd ./Angular
cd ./angular-personas
ng serve --open

Siguiendo las indicaciones anteriores, se podrá utilizar la aplicación.
